Part 3: Applying to a Project (Application)

Project Context

This final part focuses on Application—putting your design system into practice and understanding how colors behave in different environments (Digital vs. Print).

Task: Choose Your Format

Apply your 60-30-10 palette from Part 2 to a mock design. Choose one format:

Option A: Website Design (RGB)

Focus: Vibrant, digital-first colors optimized for screens.

What to Design: A single-page Website Landing Page layout including:

  • A hero section with background color, headline & sub-headline
  • A navigation bar with menu links
  • A prominent CTA button (e.g., "Get Started", "Learn More")
  • A footer section

Key Tips: Use Primary (60%) for backgrounds, Secondary (30%) for nav/footer, Accent (10%) for CTA buttons only. Use Affinity Designer to create your layout.

Option B: Poster Design (CMYK)

Focus: Print-ready, color-accurate professional poster.

What to Design: An 18" × 24" Event Poster layout including:

  • A large background area with your primary color
  • An event title block with your secondary color
  • A date/time/venue box with your accent color
  • Supporting text for event details

Key Tips: Convert HEX to CMYK. Watch for out-of-gamut warnings. Use rich black (C:40 M:40 Y:40 K:100). Use Affinity Publisher or Affinity Designer.

📋 Step-by-Step Guide

For Option A (Website — RGB) using Affinity Designer

  1. Open Affinity Designer and create a new document with an RGB color profile.
  2. Set canvas size to 1440×900 px (standard desktop).
  3. Block out the layout in grayscale first — use the Rectangle tool to draw shapes for hero, navbar, CTA, footer.
  4. Apply your 60-30-10 palette from Part 2:
    • Hero background → Primary (60%)
    • Navbar & footer → Secondary (30%)
    • CTA button → Accent (10%)
  5. Add placeholder text using the Artistic Text or Frame Text tool for headline, sub-headline, and menu items.
  6. Check readability — zoom out and squint. Can you spot the CTA?
  7. Export as JPEG or PNG (at least 1440px wide).

For Option B (Poster — CMYK) using Affinity Publisher

  1. Open Affinity Publisher (or Affinity Designer) and create a new document with a CMYK/8 color profile.
  2. Set canvas size to 18" × 24" at 300 DPI.
  3. Block out the layout — sketch the title area, background, detail boxes.
  4. Apply your 60-30-10 palette from Part 2:
    • Background → Primary (60%)
    • Title block → Secondary (30%)
    • Date/venue box → Accent (10%)
  5. Use Affinity's built-in color picker to enter your HEX values — the software auto-converts them to CMYK. Check for gamut warnings in the color panel.
  6. Check for gamut issues — look for the gamut warning indicator in Affinity's color picker. Note any adjustments.
  7. Export as PDF (File → Export → PDF, select PDF/X-1a or PDF/X-4 preset for print-ready CMYK).

  1. Mock Design File: A high-res JPEG, PNG, or PDF of your layout.
  2. Color Specification Table: List each color with its Hex code, role (60/30/10), and where it's applied.
  3. Design Rationale: 150 words explaining your color adjustments for RGB or CMYK.
